#Intro suspense image

As long as you have got a promise, then you can suspend on it.
Here is a function that turns loading an image into a promise.
This means the old image will be there fading until the new image is ready to be printed and AFTER the get request data fetched is ready

To test it do hard reload on the browser and clean all in the tab network and choose 3G (slow internet)

```html
<img src="/some/image.jpg" />
```

```tsx
function preloadImage(src: string) {
  return new Promise<string>(async (resolve, reject) => {
    const img = new Image();
    img.src = src;
    img.onload = () => resolve(src);
    img.onerror = reject;
  });
}
```

```tsx
// Outside Component
function Img({ src = '', ...props }: React.ComponentProps<'img'>) {
  src = use(imgSrc(src));
  return <img src={src} {...props} />;
}
```

From Utils

```tsx
// Do Need to have a image cache, because we do not want to be creating these this image in every render
const imgCache = new Map<string, Promise<string>>();

// Exporting to import on Component file
export function imgSrc(src: string) {
  const imgPromise = imgCache.get(src) ?? preloadImage(src);
  imgCache.set(src, imgPromise);
  return imgPromise;
}

}
```
